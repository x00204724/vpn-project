Project Overview

Modern organisations require secure, reliable and scalable connectivity to support remote & hybrid distributed teams and cloud-based applications. This project evaluates VPNs, SD-WAN, VDI, and cloud/SaaS solutions in terms of security, speed, cost, scalability, and ease of management for startups, SMEs, and enterprises.

Project Objectives

  • Design and implement a simulated enterprise network using GNS3
  • Configure and test GRE, IPSec, SSL and OpenVPN tunnels
  • Measure tunnel performance using file transfer metrics
  • Analyse packet behaviour using Wireshark
  • Implement a hybrid cloud VPN using Azure
  • Compare suitability for startups, SMEs and enterprises

Industry Context

During internship at Department of Agriculture, Food and the Marine (DAFM), IPSec VPNs and Citrix VDI used to securely connect 200+ sites. Assisted with network troubleshooting, Cisco switch setup, remote staff support.

Performance Testing & RTT Analysis

Performance testing includes measuring round-trip time and verifying GRE and IPSec tunnel operation using packet captures. RTT variations over 20 ICMP packets show stable latency across the GRE tunnel under normal operation and the impact of tunnel shutdown on connectivity.

Conclusion

This project compared various VPN solutions using GNS3 simulation and real baseline measurements. The results show encryption overhead and suitability for different organization sizes.
